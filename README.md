## Description

Backend of Markit Blog Application

## Installation

```bash
$ npm install
```

## Optional Installation

VSCode Extionsons:

- [MongoDB](https://marketplace.visualstudio.com/items?itemName=mongodb.mongodb-vscode)
- [ThunderClient](https://marketplace.visualstudio.com/items?itemName=rangav.vscode-thunder-client)

## Running the app

1. Create a `.env` file based on `.env.example`
2. Check port in `src/main.ts` and in Env in Thunder Client
3. Run the app:

```bash
# development
$ npm run start

# no watch mode
$ npm run start:nowatch

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
